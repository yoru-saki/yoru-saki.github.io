## 开始之前：先认识"终端"

我们全程会用到一个叫**终端**的小工具。它就是电脑里一个可以"打字给电脑下命令"的黑框框——
- Windows：在开始菜单搜索 **PowerShell** 或 **终端**，点开即可；
- macOS：在"启动台"里搜索 **终端（Terminal）**，点开即可。

下面凡是灰色框里的内容，都是**复制进终端、按回车执行**的命令。看不懂没关系，照做就行。

## 一、Claude Code 安装及使用

### 1.1 它是什么

你可以把 Claude Code 理解成一个"住在终端里的 AI 助手"。你用大白话告诉它要做什么，它就能帮你读文件、写代码、改内容、跑程序。你不需要自己敲每一行代码，只负责提需求、看结果、做决定。

新手如果只想先上手一个工具，推荐从 Claude Code 开始：用起来简单，对国内常见的 AI 模型也支持得比较好，配合后面讲的 CC Switch 还能随时换模型。

### 1.2 安装步骤（Windows / macOS / Linux）

俗话说"工欲善其事，必先利其器"，我们先把地基打好。

**第一步：先装两个基础工具——Git 和 Node.js**

这两个是 AI 编程工具运行的"底座"，装一次以后一直能用，你不用去理解它们具体是什么。

Windows 用户，把下面两行**分别**复制进终端、各按一次回车：

```bash
winget install Git.Git
winget install OpenJS.NodeJS
```

macOS 用户（需先装好 Homebrew）执行：

```bash
brew install git node
```

Linux 用户（以 Ubuntu / Debian 为例）执行：

```bash
sudo apt update
sudo apt install -y git nodejs npm
```

装完后，逐行输入下面的命令验证。只要每行都能显示出一串版本号（而不是报错），就说明装好了：

```bash
git --version
node -v
npm -v
```

**第二步：安装 Claude Code 本体**

推荐用下面这行命令安装（通用、最省事）：

```bash
npm install -g @anthropic-ai/claude-code
```

Windows 用户也可以改用系统自带的工具一键安装：

```bash
winget install Anthropic.ClaudeCode
```

装好后，在终端输入 `claude` 回车，如果能进入 Claude Code 的界面，就说明本体装成功了。

> 提示：第一次进去后，它还没接上可用的模型，暂时先不用急着对话。等走完第二部分、用 CC Switch 配好模型，就能正式开始使用了。

### 1.3 如果输入命令提示"找不到"（配置 PATH 路径）

有时装完后，在终端输入 `claude`、`node` 或 `npm`，会提示 **"不是内部或外部命令"**。这不是装坏了，只是电脑还不知道去哪找它们。我们手动告诉电脑一下路径即可（下面主要针对 Windows）。

**情况一：`node` 找不到——补 Node.js 的路径**

1. 先找到 Node.js 的安装文件夹：用官方安装包装的话，通常就是 `C:\Program Files\nodejs\`（如果你安装时改过位置，就用你当时选的那个文件夹）；
2. 右键"此电脑" → 属性 → 高级系统设置 → 环境变量；
3. 在下方"系统变量"里找到名为 **Path** 的一项，双击打开；
4. 点"新建"，把上面那个文件夹路径粘进去，一路点"确定"保存；
5. **关掉终端再重新打开**，输入 `node -v` 验证，能显示版本号就成功了。

**情况二：`claude` 找不到——补 npm 的路径**

用命令装的工具（比如 `claude`）会被放进 npm 的一个专门文件夹里，把这个文件夹也加进 Path 即可：

1. 在终端输入 `npm config get prefix` 回车，会显示一个文件夹路径（常见的是 `C:\Users\你的用户名\AppData\Roaming\npm`）；
2. 和上面一样，打开"环境变量 → Path → 新建"，把这个路径粘进去，保存；
3. **重新打开终端**，输入 `claude --version` 验证。

> 一句话记住：`node` 找不到就补 **Node.js 的文件夹**，`claude` 找不到就补 **npm 的文件夹**。改完路径后一定要**关掉终端重新打开**才会生效。

### 1.4 日常最常用的三个命令

装好之后，天天要用的其实就几个。在和 Claude Code 对话时，直接输入以斜杠 `/` 开头的命令再回车就行。先记住下面这三个，入门足够了。

**`/model`——换模型**

不同的活儿适合不同的 AI 模型：做整体规划、想复杂问题时用聪明一点的模型，写些简单内容时用便宜快速的模型更划算。对话中输入 `/model`，就能在**你已经配好的几个模型之间**随时切换，不用退出重来。

> 简单区分一下：**CC Switch** 负责"把模型接进来"（填地址、密钥，一次性配置好，见第二部分）；`/model` 负责在用的时候"临时换着用"。

**`/compact`——精简对话（压缩上下文）**

"上下文"你可以理解成 AI 的"短期记忆"。聊得越久，它要记的东西越多，不但反应变慢，还容易忘掉前面说过的话。这时输入 `/compact`，它会把之前聊过的内容浓缩成一小段摘要：重点留下、废话清掉，腾出记忆空间继续顺畅地聊。如果担心某些内容被删掉，可以在命令后面补一句，告诉它"重点保留 XXX"。

**`/rewind`——反悔、撤回（回到上一步）**

让 AI 干活时，它偶尔会"自信满满"地改错东西。别慌——输入 `/rewind`，就能回到之前某个时间点的状态，把最近这几步改动撤销掉，对话和文件都能退回去。可以把它当成软件自带的"后悔药"，有了它，你才敢放心大胆地让 AI 去试。

### 1.5 国产模型推荐

Claude Code 本身只是个"外壳"，真正干活的是背后的 AI 模型。下面三款国产模型的编程能力都不错，各有各的长处，可以按自己的需要挑：

**DeepSeek-V4**

深度求索出品，擅长"记性好"和"想得深"。它能一次性记住很长很长的内容，适合帮你做整体规划、搭建项目框架、处理复杂改动；还分"快速版"和"深度思考版"两种模式，按需选用，用起来很划算。

**智谱 GLM-5.1**

智谱 AI 出品，中文理解和写代码都很均衡，你说什么它照做的稳定性好，日常常规任务都能稳稳接住，是一款综合表现很全面的模型。

**Kimi K2.6**

月之暗面出品，长处是"能看图"和"记性长"。在处理和图片、页面排版有关的任务时特别出色，也能胜任日常写代码和帮忙检查代码；内容再长也不容易"读串"。

## 二、CC Switch 安装及使用

### 2.1 它是什么

CC Switch 是一个"模型切换器"。有了它，你不用手动去改一堆复杂配置，点几下就能给 Claude Code 接上、并换成你想用的模型。它不只服务 Claude Code，其他 AI 编程软件也能用。

在这套教程里，我们正是**靠它来给 Claude Code 配好第一个能用的模型**，所以这一步建议一定要完成。

### 2.2 去官网下载安装

打开下面的官网，下载适合你系统（Windows / macOS）的安装包，像装普通软件一样双击安装即可：

- 官网地址：<https://cc-switch.cc/>

装好后打开 CC Switch，就可以开始配置了。

### 2.3 配置教学（照着填就行）

打开 CC Switch，新增或编辑一个配置，按下面的顺序填写：

1. **请求地址**：填写 `https://newapi.dzkjm.cn`
2. **API Key**：填写发放给你的那串密钥（相当于"钥匙"，复制粘贴进去即可）
3. 点开下方的 **高级选项**
4. 点击 **获取模型列表**
5. 稍等片刻，列表会自动列出可用的模型，从中**选中你想用的那个**即可

填完保存，在 CC Switch 里切换到这个配置，Claude Code 就会自动用上你选好的模型了。到这里，整套工具就配置完成，可以正式开始使用啦。
